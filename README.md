# Medisync

Application for health that digitizes, automates and simplifies the organization of managers in the generation of schedules.

## Clone Repository

```bash
git clone https://github.com/bastos-rcd/MEDISYNC.git
cd MEDISYNC
```

## Landing Page

> **React** : [medisync-landing.vercel.app](https://medisync-landing.vercel.app/)

```bash
cd landing-page
# Install dependencies
npm install
# Run the application
npm run dev
# Build the application
npm run build
```
